Homework & Labs

Weight: 35% of the course grade (9 assignments).

Programming assignments in Python using TensorFlow or PyTorch. Assignments are cumulative and scaffold toward the capstone; case studies and security audits are embedded within the coding projects. The sequence follows the three pillars: AI for Security detectors first (HW1–HW4), then Security of AI attacks and defenses (HW5–HW8), then Trustworthy AI auditing and privacy (HW8–HW9). Each is graded on correctness of implementation, code quality, documentation, and adherence to the code submission standards (GitHub with README, inline comments, reproducibility, requirements.txt/environment.yml, random seeds).

Late policy: Late work is penalized 10% per day for up to 48 hours; work submitted more than 48 hours late receives no credit unless prior arrangements are made.
